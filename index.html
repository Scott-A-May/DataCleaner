// What it does

Turns messy CSVs into clean, documented Excel files

DataCleaner is a Python automation script that takes a raw CSV file and runs it through a full cleaning pipeline — fixing column names, trimming whitespace, standardizing casing, removing duplicates, converting numeric types, and handling missing values.

The output is a structured Excel workbook with three tabs: cleaned data, a full log of every change made, and a summary stats sheet — giving you a complete audit trail.

🏷️

Column Standardization

Strips whitespace, lowercases, and converts spaces to underscores in all column headers.

✂️

Whitespace Trimming

Removes leading and trailing spaces from every text cell across the entire dataset.

🔤

Text Casing

Normalizes name and state fields to Title Case for consistent formatting.

🔢

Type Conversion

Converts numeric columns stored as text (e.g. income, home_value) to proper number types.

🗑️

Duplicate Removal

Detects and drops exact duplicate rows, logging how many were removed.

🩹

Missing Value Handling

Fills numeric nulls with column median; fills text nulls with "Unknown".
